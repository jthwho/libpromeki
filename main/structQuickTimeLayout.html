Detailed Description

Well-known Enum type for QuickTime / ISO-BMFF container layout.

Used as the value type for the MediaConfig::QuickTimeLayout config key. Classic writes a traditional movie atom ending in a single moov atom; Fragmented writes an initial moov followed by a series of moof / mdat fragment pairs, which is what streaming and live ingest pipelines need.
