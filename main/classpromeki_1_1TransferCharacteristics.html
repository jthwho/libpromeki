Detailed Description

Well-known Enum type for VUI / container transfer characteristic (opto-electronic transfer function).

Numeric values match ISO/IEC 23091-4 / ITU-T H.273. This covers all of the common SDR and HDR curves: BT709 (Rec.709 gamma), SMPTE2084 (PQ, for HDR10 / HDR10+ / Dolby Vision base layer), and ARIB_STD_B67 (HLG).
