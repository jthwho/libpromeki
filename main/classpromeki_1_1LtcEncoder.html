Detailed Description

Encodes timecode values into LTC (Linear Timecode) audio samples.

Wraps libvtc's VtcLTCEncoder to produce biphase-mark-encoded audio representing SMPTE timecode. The output is mono int8_t audio samples suitable for mixing into an audio stream or output on a dedicated channel.

Not copyable (owns encoder state). Movable.

Example
LtcEncoder enc(48000, 0.5f);
Timecode tc(Timecode::NDF24, 1, 0, 0, 0);
List<int8_t> samples = enc.encode(tc);
// samples contains ~2000 int8_t samples of LTC for this frame
