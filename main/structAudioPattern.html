Detailed Description

Well-known Enum type for audio test pattern generator modes.

Mirrors AudioTestPattern::Mode in value and order. Used as the value type for the MediaIOTask_TPG ConfigAudioMode config key.

Note that the config key is historically named ConfigAudioMode, but the corresponding Enum type is called AudioPattern for consistency with VideoPattern.
