Detailed Description

Typed handle for a concrete codec backend.

Instances are obtained via VideoCodec::registerBackend or VideoCodec::lookupBackend — callers never hold the raw backend name as a String after registration. The handle is a cheap value type (one uint64_t under the hood) backed by a process-wide VideoCodecBackendRegistry.
