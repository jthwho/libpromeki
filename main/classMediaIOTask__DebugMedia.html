Detailed Description

MediaIO backend that captures a Frame into a PMDF debug file.

Thin adapter around DebugMediaFile that plugs PMDF into the MediaIO command set. Supports both MediaIO::Source (read frames back) and MediaIO::Sink (capture to disk) — the point of PMDF is lossless frame round-trip for debugging, so both modes are first-class.

Registered with the "pmdf" file extension so mediaplay -d capture.pmdf routes here automatically.
