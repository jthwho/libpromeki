Detailed Description

Well-known Enum type for InspectorMediaIO test selection.

Element type for the MediaConfig::InspectorTests EnumList — the inspector consumes a list of tests to run. An empty list runs the default suite (every value below); a non-empty list runs exactly the listed tests and disables the rest.

  • ImageData — decode the ImageDataEncoder bands carried in the picture (frame number, stream ID, picture TC).
  • Ltc — decode LTC from the audio track.
  • TcSync — picture TC vs audio LTC offset in samples. Implies ImageData + Ltc.
  • Continuity — frame number / stream ID / TC continuity. Implies ImageData.
  • Timestamp — per-essence MediaTimeStamp existence check, frame-to-frame delta (min / max / avg), and actual observed FPS.
  • AudioSamples — per-frame audio sample count (min / max / avg) plus measured audio sample rate derived from cumulative samples and audio MediaTimeStamps.
  • CaptureStats — write a per-frame TSV record (timestamps, image and audio formats, buffer sizes) to the file named by MediaConfig::InspectorStatsFile (or a unique file in Dir::temp() when that key is empty).
