Detailed Description

MediaPipeline composition layer over MediaIO.

promeki::MediaPipeline composes one or more promeki::MediaIO instances and the promeki::MediaIOPortConnection objects between them under a single lifecycle (run / pause / close). promeki::MediaPipelinePlanner handles auto-bridge insertion for format-mismatched stages.
