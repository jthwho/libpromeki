Detailed Description

Media processing pipeline: graph, nodes, and scheduling.

MediaNode and MediaPipeline framework for constructing audio/video processing pipelines. Includes concrete nodes such as TestPatternNode, FrameDemuxNode, and TimecodeOverlayNode.
