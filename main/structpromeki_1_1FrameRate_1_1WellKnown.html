Detailed Description

One named entry in FrameRate::wellKnownRates.

label is a human-readable form ("23.98", "29.97", "60", etc.) suitable for a dropdown UI; rate is the canonical FrameRate value. The label is intentionally label-style (not just the numeric form) so future entries can carry context like "(NTSC)" without polluting the Rational printer.
