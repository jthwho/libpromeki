Detailed Description

template<typename T, size_t NumValues>
class promeki::Array< T, NumValues >

Fixed-size array container wrapping std::array.

Example
Array<float, 3> rgb = {0.5f, 0.8f, 1.0f};
float r = rgb[0]; // 0.5
size_t n = rgb.size(); // 3
rgb.fill(0.0f); // all zeros
