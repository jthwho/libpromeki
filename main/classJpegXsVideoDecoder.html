Detailed Description

JPEG XS VideoDecoder built on SVT-JPEG-XS.

Symmetric counterpart to JpegXsVideoEncoder. Registered against VideoCodec::JPEG_XS. The decoder's output PixelFormat is set via MediaConfig::OutputPixelFormat; when unset, the natural planar target matching the incoming bitstream (bit depth × subsampling) is produced.
