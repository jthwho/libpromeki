Detailed Description

Describes a concrete audio stream — format plus sample rate and channel count.

AudioDesc is to AudioFormat what ImageDesc is to PixelFormat: AudioFormat identifies the sample encoding (PCM layout, bit depth, endianness, optional compressed codec) while AudioDesc binds that format to a specific sample rate, channel count, and optional metadata.

Example
AudioDesc desc(AudioFormat::PCMI_S16LE, 48000.0f, 2);
assert(desc.isValid());
assert(desc.bytesPerSample() == 2);
assert(desc.bufferSize(1024) == 4096);

◆ fromSdp()

static AudioDesc AudioDesc::fromSdp ( const SdpMediaDescription &  md)
static

Derives an AudioDesc from an SDP media description.

Interprets the RTP payload encoding name in the a=rtpmap attribute to build an audio AudioDesc. Supported encodings (all RFC 3551 / RFC 3190):

  • L16 → AudioFormat::PCMI_S16BE (wire is big-endian 16-bit)
  • L24 → AudioFormat::PCMI_S24BE (wire is packed 24-bit big-endian)
  • L8 → AudioFormat::PCMI_U8 (unsigned 8-bit per RFC 3551 §4.5.10)

Anything else (compressed codecs, non-audio media descriptions) yields an invalid AudioDesc.

Parameters
mdThe SDP media description to interpret.
Returns
A populated AudioDesc, or an invalid AudioDesc on any failure.

SdpMediaDescription AudioDesc::toSdp ( uint8_t  payloadType) const

Builds an SDP media description from this AudioDesc.

The inverse of fromSdp. Populates the returned SdpMediaDescription with m=audio, an a=rtpmap line for the appropriate encoding (L16, L24, or L8), and the clock rate / channel count derived from this descriptor.

Returns an empty SdpMediaDescription if the AudioDesc is invalid or its AudioFormat has no RTP encoding mapping.

Parameters
payloadTypeRTP payload type (0-127).
Returns
A populated SdpMediaDescription, or an empty one on failure.
