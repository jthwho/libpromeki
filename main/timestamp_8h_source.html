249// ---------------------------------------------------------------------------
250// promeki::Duration interop
251//
252// promeki::Duration and TimeStamp::Duration are intentionally different
253// types — promeki::Duration is the library's portable nanosecond-precision
254// value object, while TimeStamp::Duration is the platform's
255// std::chrono::steady_clock::duration. These free function overloads let
256// callers do TimeStamp + promeki::Duration arithmetic directly, without
257// the lossy round-trip through @c secondsToDuration / @c toSecondsDouble
258// that earlier code had to fall back on.
259//
260// Implemented as free functions so the parameter type can be the
261// namespace-level @c promeki::Duration without colliding with the
262// in-class @c TimeStamp::Duration typedef.
263// ---------------------------------------------------------------------------
