240// ---------------------------------------------------------------------------
241// promeki::Duration interop
242//
243// promeki::Duration and TimeStamp::Duration are intentionally different
244// types — promeki::Duration is the library's portable nanosecond-precision
245// value object, while TimeStamp::Duration is the platform's
246// std::chrono::steady_clock::duration. These free function overloads let
247// callers do TimeStamp + promeki::Duration arithmetic directly, without
248// the lossy round-trip through @c secondsToDuration / @c toSecondsDouble
249// that earlier code had to fall back on.
250//
251// Implemented as free functions so the parameter type can be the
252// namespace-level @c promeki::Duration without colliding with the
253// in-class @c TimeStamp::Duration typedef.
254// ---------------------------------------------------------------------------
