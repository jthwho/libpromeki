Detailed Description

JPEG VideoEncoder built on libjpeg-turbo.

Encodes uncompressed images (RGB8, RGBA8, YCbCr 4:2:2 YUYV/UYVY/planar, YCbCr 4:2:0 planar/NV12) to JPEG compressed form. Each call to submitFrame runs one JPEG encode through libjpeg-turbo and queues the resulting compressed bytes as a MediaPacket; receivePacket pops the head of that queue.

Default subsampling is 4:2:2 for RFC 2435 RTP compatibility.

Registered against VideoCodec::JPEG. Every emitted packet is flagged MediaPacket::Keyframe because every JPEG bitstream is independently decodable.

void JpegVideoEncoder::configure ( const MediaConfig &  config)
overridevirtual

Applies encoder parameters from a MediaConfig.

Each backend reads the well-known keys it understands (BitrateKbps, VideoRcMode, GopLength, VideoPreset, …) and silently ignores the rest. Calling configure() while a session is active is allowed; the backend applies what it can at runtime and defers the rest to the next IDR. The default implementation is a no-op.
