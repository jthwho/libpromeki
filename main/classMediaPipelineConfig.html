Detailed Description

Holds the complete configuration for building a MediaPipeline.

MediaPipelineConfig contains a list of MediaNodeConfig objects, one per node to create. The list need not be in any particular order; the pipeline determines dependency order during build().

Example
MediaPipelineConfig pipeConfig;
MediaNodeConfig src("TestPatternNode", "source");
src.set("Width", uint32_t(1920));
pipeConfig.addNode(src);
MediaNodeConfig sink("RtpVideoSinkNode", "output");
sink.addConnection("source[0]");
pipeConfig.addNode(sink);
auto result = pipeline.build(pipeConfig);
if(result.isError()) { ... }
