Constructs a MediaIO writer that plays frames through SDL using the legacy FramePacer path.

Deprecated:
Use createSDLPlayer instead — see SDLPlayerOldTask.

Creates an SDLPlayerOldTask with the supplied widget/audio pointers and adopts it into a newly allocated MediaIO. The returned MediaIO is not yet open; the caller must set the source MediaDesc (and optionally an AudioDesc) and then call open(MediaIO::Input).

The returned MediaIO owns the task and deletes it on destruction. Ownership of video and audio stays with the caller — they must outlive the returned MediaIO.

Parameters
videoVideo widget to display frames in (may be nullptr).
audioAudio output for playback (may be nullptr).
useAudioClocktrue to prefer the audio device as the timing source (default); false to use wall clock.
parentOptional parent object for the MediaIO.
Returns
A new MediaIO ready to be configured and opened as a writer, or nullptr if task adoption fails.
