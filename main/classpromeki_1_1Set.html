Detailed Description

template<typename T>
class promeki::Set< T >

Ordered unique-element container wrapping std::set.

Provides a Qt-inspired API over std::set with consistent naming conventions matching the rest of libpromeki.

Thread Safety
Conditionally thread-safe. Distinct instances may be used concurrently; concurrent access to a single instance must be externally synchronized.
Template Parameters
TElement type (must support operator<).
Example
Set<String> tags = {"alpha", "beta"};
tags.insert("gamma");
bool has = tags.contains("alpha"); // true
tags.remove("beta");
List<String> sorted = tags.toList();
