Detailed Description

Low-level file I/O wrapper.

Provides a platform-independent interface for opening, reading, writing, and seeking within files. Supports flags for controlling access mode, direct I/O, synchronous writes, and other behaviors. This class is non-copyable.

Open in non-blocking mode.

DirectIO 

Use direct I/O, bypassing OS page cache.

Synchronous write mode.
