◆ toFrameCount()

FrameCount promeki::toFrameCount ( const FrameNumber &  n)
inline

Reinterprets a FrameNumber as a FrameCount.

This is the explicit "treat this index as a count" conversion — useful when "the index of the next frame" doubles as "the number of frames written so far". An Unknown FrameNumber returns an Unknown FrameCount.

Parameters
nThe FrameNumber to convert.
Returns
FrameCount(n.value()) for valid frame numbers, otherwise Unknown.

◆ toFrameNumber()

FrameNumber promeki::toFrameNumber ( const FrameCount &  c)
inline

Reinterprets a FrameCount as a FrameNumber.

This is the explicit "use a count as an index" conversion — typically because a counter (e.g. "I've written N frames") doubles as the index of the next frame. Unknown / Infinity FrameCounts return an Unknown FrameNumber.

Parameters
cThe FrameCount to convert.
Returns
FrameNumber(c.value()) for finite counts, otherwise Unknown.
