Detailed Description

Per-stage telemetry record.

Combines the cumulative MediaIO::stats aggregate with the per-MediaIOCommand::Kind windowed breakdown collected by MediaIOStatsCollector so a single value fully describes a stage's runtime state. Round-trips through JSON (for logging / REST endpoints) and DataStream (for IPC).

Thread Safety
Conditionally thread-safe. Distinct instances may be used concurrently; concurrent access to a single instance must be externally synchronized.

◆ windowed

WindowedMap promeki::MediaPipelineStageStats::windowed

Per-command-kind windowed breakdown.

Keyed by MediaIOCommand::Kind. Each value is a WindowedStatsBundle holding one WindowedStat per stat ID the MediaIOStatsCollector observed for that command kind (ExecuteDuration, QueueWaitDuration, BytesProcessed plus any backend-specific keys). Empty when no collector was attached or no commands of that kind have resolved yet.
