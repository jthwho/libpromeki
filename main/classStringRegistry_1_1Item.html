Detailed Description

template<typename Tag>
class StringRegistry< Tag >::Item

Lightweight handle identifying a registered string.

Item wraps a uint32_t ID and provides access to the underlying string via the owning registry. Items are cheap to copy and can be compared by integer value.

Warning
The integer ID is assigned at registration time and must not be persisted or used outside the current process.
