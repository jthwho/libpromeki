Detailed Description

Well-known Enum type for audio sample formats.

Mirrors AudioDesc::DataType in value and order. Used as the value type for any config key that selects an audio sample format (e.g. MediaConfig::OutputAudioDataType).

The integer values match the AudioDesc::DataType enumeration so callers can convert in either direction via a plain static_cast on Enum::value(). The string names also match — e.g. "PCMI_S16LE", "PCMI_Float32LE" — so legacy code that still passes strings through AudioDesc::stringToDataType keeps working when the same string is fed through the Enum lookup path.
