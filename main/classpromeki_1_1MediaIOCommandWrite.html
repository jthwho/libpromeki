◆ group

MediaIOPortGroup* promeki::MediaIOCommandWrite::group = nullptr

The port group this write targets.

Set by MediaIOSink::writeFrame from the sink's owning group. Backends use it to know which group's sink(s) the frame(s) feed — for sync-grouped writes this is the join key for collecting one frame per sink in the group before dispatching as a batch.

◆ sink

MediaIOSink* promeki::MediaIOCommandWrite::sink = nullptr

The originating sink port.

Set by MediaIOSink::writeFrame so MediaIO::completeCommand can route per-sink frameWanted / writeError signals back to the specific sink that submitted the write. Backends never read this — they receive frames per-group via group above.
