Detailed Description

template<typename T, size_t W, size_t H>
class Matrix< T, W, H >

Generic fixed-size matrix with compile-time dimensions.

Example
Matrix<float, 4> m = Matrix<float, 4>::identity();
m.translate(1.0f, 2.0f, 3.0f);
auto inv = m.inverted();
