Constructs a MediaIO writer that plays frames through SDL.

Creates an SDLPlayerTask with the supplied widget/audio pointers and adopts it into a newly allocated MediaIO. The returned MediaIO is not yet open; the caller must set the source MediaDesc (and optionally an AudioDesc) and then call open(MediaIO::Writer).

The returned MediaIO owns the task and deletes it on destruction. Ownership of video and audio stays with the caller — they must outlive the returned MediaIO.

Parameters
videoVideo widget to display frames in (may be nullptr).
audioAudio output for playback (may be nullptr).
pacedtrue for frame-rate enforced playback (default), false for as-fast-as-possible with audio dropped.
parentOptional parent object for the MediaIO.
Returns
A new MediaIO ready to be configured and opened as a writer, or nullptr if task adoption fails.
