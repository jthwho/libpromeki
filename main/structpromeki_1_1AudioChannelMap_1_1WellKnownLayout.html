Detailed Description

One entry in the well-known layout dictionary.

name is the canonical layout name accepted by fromString and emitted by toString / wellKnownName when this layout matches.

roles is the canonical role list — well-known layouts are stream-agnostic; the stream is selected at instantiation time.
