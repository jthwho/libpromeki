Detailed Description

Immutable data record for an audio format.

Populated by the library for well-known formats, or by applications via registerData for custom formats. The sample-level fields (bytesPerSample, bitsPerSample, isSigned, isFloat, isPlanar, isBigEndian) only have meaningful values for PCM formats; compressed formats set compressed to true and leave those fields at defaults.
