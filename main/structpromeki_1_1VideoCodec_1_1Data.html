Detailed Description

Immutable descriptor for a video codec.

Populated by the library for well-known codecs, or by applications via registerData for custom codecs. Captures codec-family facts that do not depend on any specific backend implementation — per-backend capabilities live on the VideoEncoder / VideoDecoder backend records.
