Detailed Description

TCP server that listens for incoming connections.

TcpServer binds to a local address and listens for incoming TCP connections. Accepted connections are returned as TcpSocket objects.

Thread Safety
Inherits ObjectBase — thread-affine. A single TcpServer must only be used from the thread that created it.
Example
TcpServer server;
server.listen(SocketAddress::any(8080));
server.waitForNewConnection(5000);
TcpSocket *client = server.nextPendingConnection();

◆ nextPendingDescriptor()

int promeki::TcpServer::nextPendingDescriptor ( )

Drains the next pending connection as a raw fd.

Equivalent to nextPendingConnection except the caller takes the bare descriptor and is responsible for closing it. Useful when the caller wants to wrap the descriptor in something other than a plain TcpSocket — for example an SslSocket that needs to drive a TLS handshake on top.

Returns
The accepted fd, or -1 when no connection is pending or accept() failed.
