Detailed Description

A versatile string class inspired by Qt's QString.

Internally wraps std::string but provides a less verbose interface with additional convenience methods. Implicit conversion operators allow passing a String directly to functions expecting std::string or const char*. For std::string member functions not directly exposed, use stds() to obtain a reference to the underlying std::string.
