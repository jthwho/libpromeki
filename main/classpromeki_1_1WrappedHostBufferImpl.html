Detailed Description

Non-owning HostMappedBufferImpl over an external host pointer.

Used by Buffer::wrapHost to expose externally-managed memory (mmap regions, hardware DMA staging, stack arrays) through the BufferImpl interface without taking ownership. The destructor is a no-op; the caller retains lifetime responsibility for the underlying allocation.

canClone returns false — wrapping memory we do not own makes "clone" meaningless. Buffer::ensureExclusive on a wrapped buffer surfaces Error::NotSupported rather than detaching.
