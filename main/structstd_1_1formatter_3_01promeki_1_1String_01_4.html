Detailed Description

std::formatter specialization for promeki::String.

Inherits from std::formatter<std::string_view> so that the full set of standard string format specifiers (width, fill, alignment, precision) is available for free. The String's byte representation is exposed as a string_view via cstr() / byteCount() — for Unicode-encoded Strings this is the cached UTF-8 byte sequence, which is the right thing to feed into std::format.
