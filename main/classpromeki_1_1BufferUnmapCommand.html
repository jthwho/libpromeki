Detailed Description

Command to release a previously-acquired domain mapping.

Built by Buffer::mapRelease. The backend decrements the per-domain refcount and, when the count drops to zero, performs any required write-back / unmap. When the buffer was never mapped to target, the backend completes the command with Error::Invalid.
