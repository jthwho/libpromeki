Detailed Description

Input endpoint on a MediaNode that receives Frame::Ptr.

Each MediaSink owns a thread-safe Queue for buffering incoming frames and reports backpressure via the virtual canAcceptFrame() method. When a frame is pushed, the owning node is woken so it can process.

MediaSink is managed via SharedPtr for lifetime management. It is an identity object (non-copyable) – the clone method should never be called.
