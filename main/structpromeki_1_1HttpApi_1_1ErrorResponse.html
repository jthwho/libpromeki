Detailed Description

One declared error response.

OpenAPI requires every status code an endpoint can return to be declared; this struct is what the catalog publishes for each. The default policy (see setDefaultErrors) installs a sensible 400 / 404 / 500 set on every endpoint that doesn't override it.
