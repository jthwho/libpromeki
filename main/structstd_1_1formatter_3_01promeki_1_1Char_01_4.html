Detailed Description

std::formatter specialization for promeki::Char.

Renders the character as its UTF-8 byte sequence (1–4 bytes). All standard string format specifiers (width, fill, alignment) are supported via inheritance from std::formatter<std::string_view>. To format the integer codepoint instead, pass c.codepoint() as an unsigned integer.
