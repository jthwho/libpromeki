std::formatter specialization for promeki::Duration.

Format spec syntax
{} // default HMS e.g. "1h 23m 45.123s"
{:hms} // explicit HMS
{:scaled} // auto-scaled e.g. "1.5 ms"

Standard string format specifiers (width, fill, alignment) follow the style keyword after a colon:

{:scaled:>16} // auto-scaled, right-justified, width 16
