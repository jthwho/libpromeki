Detailed Description

MediaIOTask backend that transforms media frames.

Generic ReadWrite MediaIO that accepts a frame on writeFrame(), applies stateless intra-frame transforms, and emits the result on readFrame(). Supported transforms in this initial version:

  • Video colorspace conversion (CSC) — delegated to Image::convert() via the CSC framework. Applies whenever ConfigOutputPixelDesc is set and both source and target pixel descriptions are uncompressed.
  • JPEG encode — delegated to JpegImageCodec::encode(). Applies when ConfigOutputPixelDesc is a compressed JPEG pixel description and the source image is uncompressed. Quality and chroma subsampling are taken from ConfigJpegQuality and ConfigJpegSubsampling.
  • JPEG decode — delegated to JpegImageCodec::decode(). Applies when the source image is a compressed JPEG and ConfigOutputPixelDesc is an uncompressed pixel description.
  • Audio sample-format conversion — delegated to Audio::convertTo(). Applies whenever ConfigOutputAudioDataType is set and differs from the source data type.

If no output keys are set, the Converter is a pass-through: the input frame is delivered unchanged on the next readFrame(). Images and audio for which no corresponding output key is set are forwarded as-is, so a Converter with only an audio config leaves video untouched and vice-versa.

The Converter is stateless with respect to temporal codecs: each writeFrame() produces exactly one output frame. Stateful encoders (H.264, HEVC, etc.) are deferred to a later revision.

Mode support

Only MediaIO::ReadWrite is supported. The Converter has no independent input source, so Reader mode has nothing to read; and no independent output sink, so Writer mode has nowhere to write.

Back-pressure

The Converter maintains an internal output FIFO. When the FIFO is full, executeCmd(MediaIOCommandWrite&) returns Error::TryAgain so the producer can back off. When the FIFO is empty, executeCmd(MediaIOCommandRead&) returns Error::TryAgain so the consumer can wait for more input. FIFO capacity is configurable via ConfigCapacity (default 4).

Example
MediaIO::Config cfg;
cfg.set(MediaIO::ConfigType, "Converter");
cfg.set(MediaIOTask_Converter::ConfigOutputPixelDesc,
PixelDesc(PixelDesc::RGBA8_sRGB));
MediaIO *io = MediaIO::create(cfg);
io->open(MediaIO::ReadWrite);
io->writeFrame(inputFrame);
Frame::Ptr outputFrame;
io->readFrame(outputFrame);
io->close();
delete io;
