Detailed Description

Wall-clock implementation of FramePacerClock.

Uses std::chrono::steady_clock for timing and std::this_thread::sleep_until for waiting. This is the default clock created by FramePacer when no external clock is supplied.

int64_t WallPacerClock::resolutionNs ( ) const
overridevirtual

Smallest meaningful time step in nanoseconds.

The resolution tells the pacer (and callers) what precision to expect. A 48 kHz audio clock has ~20 833 ns resolution (one sample period); a PTP hardware clock may report 1 ns; the built-in wall clock is ~1 ns (steady_clock).

The pacer uses this to avoid polling finer than the clock can distinguish and to qualify error measurements in the periodic debug log.

Returns
Resolution in nanoseconds (must be >= 1).

Implements FramePacerClock.
