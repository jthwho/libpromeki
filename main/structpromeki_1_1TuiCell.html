Detailed Description

A single cell in the TUI screen buffer.

Each cell holds a character and a TuiStyle describing its visual properties (foreground, background, text attributes).

Thread Safety
Conditionally thread-safe. Distinct instances may be used concurrently; concurrent access to a single instance must be externally synchronized. TuiCell is a small value type; copying is the recommended way to share it across threads.
