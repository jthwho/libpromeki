Detailed Description

template<typename T>
class Future< T >

Asynchronous result wrapping std::future<T>.

Returned by Promise<T>::future() and ThreadPool::submit().

Example
Promise<int> promise;
Future<int> future = promise.%future();
// In another thread:
promise.setValue(42);
// In the waiting thread:
int result = future.get(); // 42
