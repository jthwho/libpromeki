Detailed Description

Base class for MediaIO commands.

Commands are shared via SharedPtr<MediaIOCommand, false> with COW disabled — derived types are owned via the proxy path and never cloned. The Promise is fulfilled by the worker thread when the command completes; the caller waits on the associated Future.
