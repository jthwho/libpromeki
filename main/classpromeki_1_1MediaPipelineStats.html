Detailed Description

Telemetry snapshot for a running MediaPipeline.

Holds an array of MediaPipelineStageStats entries — one per live stage in topological order — produced by MediaPipeline::stats.

Thread Safety
Conditionally thread-safe. Distinct instances may be used concurrently; concurrent access to a single instance must be externally synchronized. MediaPipelineStats::Ptr uses an atomic refcount and is safe to share across threads.
