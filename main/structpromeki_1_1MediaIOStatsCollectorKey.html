Detailed Description

Composite key identifying a single window in MediaIOStatsCollector.

A window is uniquely identified by the MediaIOCommand::Kind that produced the sample and the MediaIOStats::ID under which the sample was stamped, so a Read command's ExecuteDuration and a Write command's ExecuteDuration end up in distinct windows even though they share an ID.

Hoisted out of MediaIOStatsCollector so that std::hash<MediaIOStatsCollectorKey> can be specialized at namespace scope before the collector instantiates HashMap<Key, ...>. MediaIOStatsCollector::Key is preserved as a type alias.
