Detailed Description

Non-owning proxy for a single slice in the list.

Entry carries a parent pointer and an index; it never owns a Buffer::Ptr. All accessors go back through the parent's buffer table and view record. Entry is cheap to copy and safe to pass by value within a single expression; do not store an Entry beyond the lifetime of its parent BufferView.

size_t promeki::BufferView::Entry::bufferIndex ( ) const

Returns the index of the backing buffer in the parent's deduplicated buffer table.

Two slices that share the same Buffer report the same bufferIndex, so callers can detect buffer sharing without comparing pointers. Returns the maximum size_t value when the entry carries no buffer.
