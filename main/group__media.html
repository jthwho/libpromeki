Detailed Description

Aligned memory buffers for media data.

The Buffer class provides page-aligned, reference-counted memory suitable for image planes, audio sample blocks, and other bulk media data.

Strongly-typed key into a MediaConfig.

Convenience alias so call sites that want to declare their own extra keys (e.g. a user-defined converter subclass) can say MediaConfigID{"MyKey"} without having to spell out the full nested type.
