Detailed Description

Server that accepts local (same-host) stream connections.

LocalServer listens for incoming connections on a path-named stream socket. On POSIX it binds an AF_UNIX / SOCK_STREAM socket at the requested filesystem path, listens, and returns accepted connections as LocalSocket instances.

The server creates its socket file on listen and unlinks it on close (or destruction). If a stale socket file from a prior crashed run exists at the requested path, listen will silently remove it before rebinding; this is safe because filesystem sockets have no inode reuse issues of their own.

This class must only be used from the thread that created it (or moved to via moveToThread).

Example
LocalServer server;
Error err = server.listen("/tmp/my-service.sock");
server.waitForNewConnection(5000);
LocalSocket *client = server.nextPendingConnection();
// caller owns the returned socket

Binds and listens at the given filesystem path.

Removes any stale socket file at path (if it is in fact a socket) before binding. Applies mode via fchmod / umask workaround and, if groupName is non-empty, chown's the socket file to that group.

Parameters
pathFilesystem path for the socket.
modePOSIX file mode for the socket file.
groupNameOptional group to chown to (empty = skip).
backlogMaximum pending-connection queue length.
Returns
Error::Ok on success, or an error.
