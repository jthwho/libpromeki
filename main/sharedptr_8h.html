◆ PROMEKI_SHARED_ABSTRACT

#define PROMEKI_SHARED_ABSTRACT (   INTERMEDIATE)
Value:
public: \
INTERMEDIATE *_promeki_clone() const override = 0;

Macro for abstract intermediate classes in a shared hierarchy.

Use this on a class that derives from a base already marked with PROMEKI_SHARED_BASE (or a hand-rolled equivalent) and that stays abstract — concrete leaves below supply the actual clone. The macro adds a covariant pure-virtual override of _promeki_clone so SharedPtr<INTERMEDIATE>::modify() detaches directly into an INTERMEDIATE pointer without a downcast, and sharedPointerCast<INTERMEDIATE>(basePtr) yields a typed Ptr whose modify() is valid without further work at the call site.

Example:

class VideoPayload : public MediaPayload {
PROMEKI_SHARED_ABSTRACT(VideoPayload)
public:
/ concrete leaves provide _promeki_clone
};

Macro for concrete leaves of a shared-object hierarchy.

Adds a covariant _promeki_clone() override that returns DERIVED* (so SharedPtr<DERIVED>::modify() detaches into a typed pointer without a downcast). The body delegates to promekiCloneOrAbort, so leaves that are themselves non-copyable compile into an abort-on-clone path instead of needing a separate macro.

Assumes the base class is already marked with PROMEKI_SHARED_BASE (or hand-rolls an equivalent virtual _promeki_clone).

Example:

class MyDerivedClass : public MyBaseClass {
PROMEKI_SHARED_DERIVED(MyDerivedClass)
public:
virtual ~MyDerivedClass();
// The rest of your class definition
};

◆ PROMEKI_SHARED_FINAL

#define PROMEKI_SHARED_FINAL (   TYPE)
Value:
public: \
RefCount _promeki_refct; \
TYPE *_promeki_clone() const { \
return ::promeki::promekiCloneOrAbort<TYPE>(this); \
}

Macro for non-polymorphic native shared objects.

Use this instead of PROMEKI_SHARED_BASE when the class will never be subclassed (e.g. private inner Data classes, plain value types). It avoids the vtable cost by using a non-virtual _promeki_clone(). The class does not need a virtual destructor.

The body delegates to promekiCloneOrAbort, so types that cannot be copied (own a Promise, UniquePtr, etc.) compile into an abort-on-clone path; pair those with SharedPtr<TYPE, false> so the abort path is never taken at runtime.

Example:

class Data {
PROMEKI_SHARED_FINAL(Data)
public:
int value = 0;
};
