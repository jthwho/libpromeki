Detailed Description

Monotonic elapsed-time measurement using std::chrono::steady_clock.

The timer starts automatically on construction. Call start() or restart() to reset. The timer can be invalidated to indicate "not running".

Thread Safety
Conditionally thread-safe. Distinct instances may be used concurrently; concurrent start / restart / invalidate on a single instance must be externally synchronized. Read accessors (elapsedMs / elapsedSeconds / isValid) are safe alongside other reads.
