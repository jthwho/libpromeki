Detailed Description

Result of running a single BenchmarkCase.

A BenchmarkResult holds timing statistics aggregated across one or more repeats of the same case at the same iteration count, plus any throughput counters and custom metrics the case reported via BenchmarkState. Results round-trip through JSON so baselines can be stored on disk.
