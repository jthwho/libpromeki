◆ group

MediaIOPortGroup* promeki::MediaIOCommandRead::group = nullptr

The port group this read targets.

Set by MediaIOSource when it builds the read command from the source's owning group. Backends use it to know which group's source(s) the produced frame(s) belong to — for sync-grouped reads this is the join key for distributing one frame per source in the group.

◆ mediaDescChanged

bool promeki::MediaIOCommandRead::mediaDescChanged = false

Set true when the backend wants to update the MediaDesc that MediaIO caches.

Used for variable-frame-rate or segmented streams whose format changes mid-playback. When true, the framework copies updatedMediaDesc into the cache, stamps Metadata::MediaDescChanged on the returned frame, and emits the descriptorChanged signal.

◆ step

int promeki::MediaIOCommandRead::step = 1

Per-group step at submit time.

Mirror of MediaIOPortGroup::step copied at submit time so the backend doesn't have to re-read the live group state during executeCmd (positive = forward, 0 = repeat / hold, negative = reverse).
