Detailed Description

Mutual exclusion lock wrapping std::mutex.

Provides a simple mutex with lock/unlock/tryLock and a nested RAII Locker type for scoped locking. Non-copyable and non-movable.

Thread Safety
Fully thread-safe by construction. All public methods may be invoked concurrently from any thread on a single instance — that is the entire purpose of the type.
Example
Mutex mtx;
{
Mutex::Locker lock(mtx);
// ... critical section ...
}
