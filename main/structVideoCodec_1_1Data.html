Detailed Description

Immutable descriptor for a video codec.

Populated by the library for well-known codecs, or by applications via registerData for custom codecs. Captures codec-family facts that do not depend on any specific backend implementation — per-backend capabilities live on the VideoEncoder / VideoDecoder backend records.

Compressed PixelFormat IDs this codec produces / consumes.

◆ rateControlModes

List<RateControlMode> VideoCodec::Data::rateControlModes

Rate-control modes the codec can be driven in.

Same shape as AudioCodec::Data::rateControlModes. Empty means "constant-quality / fixed codec" (lossless-only, JPEG at fixed quality, etc.).

◆ supportedBitDepths

List<int> VideoCodec::Data::supportedBitDepths

Bit depths the codec spec permits, in bits per component.

Empty means "no spec-level restriction" (unusual — most modern codecs enumerate 8 / 10 / 12).
