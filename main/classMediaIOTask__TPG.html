Detailed Description

MediaIOTask backend that generates test patterns.

Read-only MediaIO source that produces synchronized Frame objects containing optional video, audio, and timecode. Each component is independently enabled so the generator can serve as a video-only, audio-only, or combined source.

Video generation is delegated to VideoTestPattern, audio to AudioTestPattern, and timecode to TimecodeGenerator.

This is an infinite source: frameCount returns FrameCountInfinite and canSeek returns false.

Key Type Default Description

MediaConfig::VideoBurnFontPath String "" TrueType font path. Empty = bundled default font.
MediaConfig::VideoBurnFontSize int 0 Font size in pixels. 0 = auto-scale from image height (36px at 1080p).
MediaConfig::VideoBurnText String "{Timecode:smpte}" VariantLookup<Frame>::format template for the burn text. Resolved per-frame against the assembled Frame after all per-frame metadata has been added. Empty string disables the burn for the call. Use '\n ' inside the template to span multiple lines.
MediaConfig::VideoBurnPosition Enum BurnPosition BottomCenter Position preset.
MediaConfig::VideoBurnTextColor Color White Burn text foreground color.
MediaConfig::VideoBurnBgColor Color Black Burn text background color.
MediaConfig::VideoBurnDrawBg bool true Draw padded background rectangle behind the burn text.

The burn-in runs on top of the cached static background when the pattern is non-moving, so turning burn on is effectively free on the render side beyond one plane copy plus the text draw. The text itself comes from MediaConfig::VideoBurnText, which is treated as a VariantLookup<Frame>::format template — it is resolved against the assembled Frame after all per-frame metadata (timecode, etc.) has been written, so any registered metadata key may be referenced via {Key[:spec]}.

Example
// The default config enables video+audio+timecode, so the plain
// form is enough for a ready-to-use TPG source.
MediaIO::Config cfg = MediaIO::defaultConfig("TPG");
MediaIO *io = MediaIO::create(cfg);
io->open(MediaIO::Source);
Frame::Ptr frame;
io->readFrame(frame);
io->close();
delete io;
