Detailed Description

Path-named local (same-host) stream socket.

LocalSocket is a stream-oriented, connection-based socket for same-host inter-process communication. On POSIX it is implemented with AF_UNIX / SOCK_STREAM and identified by a filesystem path. On Windows it would be backed by a named pipe or Windows-10-era AF_UNIX; only POSIX is implemented for the MVP.

Instances derive from IODevice, so once connected they are usable with DataStream, BufferIODevice, and anything else that takes an IODevice *.

Thread Safety
Inherits IODevice — thread-affine. A single LocalSocket must only be used from the thread that created it.
Example
LocalSocket sock;
Error err = sock.connectTo("/tmp/my-service.sock");
if(err.isOk()) {
sock.write("ping", 4);
char buf[16];
int64_t n = sock.read(buf, sizeof(buf));
}
