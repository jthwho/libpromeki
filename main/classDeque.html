Detailed Description

template<typename T>
class Deque< T >

Double-ended queue container wrapping std::deque.

Provides a Qt-inspired API over std::deque with consistent naming

  • Example
    Deque<int> dq;
    dq.pushToBack(1);
    dq.pushToFront(0);
    int first = dq.front(); // 0
    dq.popFromFront();
