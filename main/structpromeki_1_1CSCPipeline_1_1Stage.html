Detailed Description

Internal stage descriptor.

Each stage holds a function pointer to its kernel and the parameters it needs. Stages are stored in a flat list and invoked sequentially for each scanline.

int promeki::CSCPipeline::Stage::semBufMap[8] = {0, 1, 2, 3, 4, 5, 6, 7}

Semantic buffer map for unpack/pack.

Maps component index to SoA buffer index based on semantic meaning (R/Y->0, G/Cb->1, B/Cr->2, A->3). Handles formats with non-standard component ordering such as BGRA, ARGB, etc.
