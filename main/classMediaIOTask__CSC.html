Detailed Description

MediaIOTask backend that performs color space conversion.

ReadWrite MediaIO that accepts a frame on writeFrame(), converts each uncompressed video image to the configured output PixelDesc via Image::convert, and emits the result on readFrame(). Audio and metadata are forwarded unchanged.

If no MediaConfig::OutputPixelDesc is set (or it is invalid), video images pass through unchanged. Compressed pixel formats are rejected — use MediaIOTask_VideoEncoder / MediaIOTask_VideoDecoder for codec transitions.

Mode support

Only MediaIO::Transform is supported.

Back-pressure

The task maintains an internal output FIFO. Writes always succeed — frames are never silently dropped. When the FIFO exceeds the configured capacity a one-shot warning is logged. When the FIFO is empty, readFrame() returns Error::TryAgain. FIFO capacity is configurable via MediaConfig::Capacity (default 4).

Example
MediaIO::Config cfg;
cfg.set(MediaConfig::Type, "CSC");
cfg.set(MediaConfig::OutputPixelDesc,
PixelDesc(PixelDesc::RGBA8_sRGB));
MediaIO *io = MediaIO::create(cfg);
io->open(MediaIO::Transform);
io->writeFrame(inputFrame);
Frame::Ptr outputFrame;
io->readFrame(outputFrame);
io->close();
delete io;
