Detailed Description

Wall-clock implementation of Clock.

Uses std::chrono::steady_clock for timing and std::this_thread::sleep_until for waiting. Its domain is ClockDomain::SystemMonotonic, its jitter envelope is near-zero, and its rate ratio is 1.0. Cannot be paused and applies no filtering by default.

Native blocking wait in raw clock nanoseconds.

Invoked by sleepUntil after domain validation and offset translation. If targetNs is in the past, implementations should return Error::Ok immediately. Implementations backed by a disappearing resource (audio device, remote clock) return the relevant error when the wait cannot complete.
