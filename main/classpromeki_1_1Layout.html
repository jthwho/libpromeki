Detailed Description

Abstract base class for layout managers.

A layout positions child widgets within the available rectangle. Layouts can be nested by adding sub-layouts. The layout system is backend-agnostic — it operates purely on geometry and size policies, making it usable by both TUI and SDL (or any other UI backend).
