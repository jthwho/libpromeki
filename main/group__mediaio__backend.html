Detailed Description

Implementer-facing classes for backend authors.

The strategy class hierarchy (promeki::CommandMediaIO, promeki::InlineMediaIO, promeki::SharedThreadMediaIO, promeki::DedicatedThreadMediaIO), the promeki::MediaIOCommand hierarchy and concrete subclasses (Open / Close / Read / Write / Seek / Params / Stats), and the promeki::MediaIOReadCache used by sources.

See the MediaIO Backend AuthorGuide" for the implementer walk-through.
