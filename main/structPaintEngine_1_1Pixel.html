Detailed Description

Opaque pixel value with inline fixed-size storage.

No heap allocation. Stores up to MaxPixelBytes bytes of format-specific pixel data. The _size field tracks how many bytes are meaningful; isEmpty() returns true when no pixel data has been written (e.g. invalid createPixel call).
