MediaIO * promeki::BurnFactory::create ( const Config &  config,
ObjectBase *  parent = nullptr 
) const
overridevirtual

Constructs a fresh MediaIO instance.

The returned object is fully wired but not yet open — callers invoke MediaIO::open afterward. Must never return nullptr unless the backend genuinely cannot allocate state to open later (e.g. an unknown sub-format selected by config). Caller takes ownership.

Parameters
