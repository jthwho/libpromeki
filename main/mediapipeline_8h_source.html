717 // Pipeline-wide frame-count cap, cached at build time
718 // from @ref MediaPipelineConfig::frameCount. Zero means
719 // "no cap" (matches the @ref FrameCount::isFinite() gate).
720 // Non-zero values are passed as the per-sink frame
721 // limit to @ref MediaIOPortConnection::addSink for every
722 // terminal-sink edge; the connection enforces the cap at
723 // safe-cut-point boundaries and emits @c sinkLimitReached
724 // when each sink hits it. @ref _terminalSinksRemaining
725 // counts the still-pending capped sinks; when it reaches
726 // zero the pipeline initiates a clean close.
