Detailed Description

Aggregated descriptive statistics computed in a single pass.

Returned by stats so callers wanting more than one of min / max / average / stddev / sum / sampleCount avoid the O(N * k) re-scan that calling each accessor in sequence would incur. The fields hold the same values the matching accessors would return — empty windows produce all zeros (and zero stddev).
