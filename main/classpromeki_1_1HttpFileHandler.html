Detailed Description

Static-file-serving HTTP handler.

HttpFileHandler maps a request path (relative to the route mount point) onto a file under a configured root Dir, then streams the file back through the response. Real on-disk paths and compiled-in ":/.PROMEKI/" resource paths are both supported because File understands both.

Behaviors:

  • Only GET (and HEAD, returning headers only) are honored; other methods produce 405 with the Allow header set.
  • Path-traversal hardening: "..", absolute, and percent-encoded "%2e%2e" segments resolve to 403.
  • MIME type lookup from a small built-in extension table; callers can register additional types via addMimeType.
  • ETag generated from size + mtime; If-None-Match yields 304.
  • Last-Modified set from the file's mtime; If-Modified-Since yields 304 when no newer modification has happened.
  • Single-range Range requests served via 206 + Content-Range.
  • Bodies are streamed via the response's HttpResponse::setBodyStream — the entire file is never held in memory at once.
  • Trailing-slash and missing-extension paths fall through to an optional index file (default "index.html").
Example
server.route("/static/{path:*}", HttpMethod::Get,
HttpFileHandler::Ptr::takeOwnership(
new HttpFileHandler(":/PROMEKI/web")));

The "{path:*}" placeholder captures everything after the mount prefix into the "path" path-param, which the handler resolves relative to its root. When mounted with no "{path}" param, the handler treats the entire request path as the relative key.

Thread Safety
Inherits HttpHandler. serve is intended to be called from the HttpServer's owning EventLoop thread. The MIME registry and root path are conceptually mutable but should be configured at startup; runtime mutation requires external synchronization.

Whether to allow directory listings when no index file matches.

Default false. When enabled, the handler emits a minimal HTML index for browser convenience. Off by default because directory listings are easy to mis-deploy as an unintended information disclosure.
