Detailed Description

template<typename T>
struct promeki::CRC< T >::Params

Configurable parameters for one CRC variant.

Mirrors the Rocksoft model with one simplification: reflect controls both input and output reflection together. All catalogued CRCs the library currently provides have refIn == refOut.
