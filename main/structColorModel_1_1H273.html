Detailed Description

H.273 / ISO/IEC 23091-4 codepoint triplet describing a color space in codec VUI terms.

Each field is the numeric value that would appear in an H.264 / HEVC VUI or AV1 color-description header. A 0 indicates "not derivable from this ColorModel" — callers typically substitute 2 (Unspecified) in that case, or fall through to an explicit user override.

Note
Transfer characteristics auto-derivation does not distinguish HDR curves (PQ / HLG) today because the library's ColorModel doesn't model them explicitly yet. HDR callers must stamp the correct transfer on the encoder config.
