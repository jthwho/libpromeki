void promeki::registerBufferImplFactory ( MemSpace::ID  id,
BufferImplFactory  factory 
)

Registers a factory function for a MemSpace ID.

Built-in MemSpace IDs (System, SystemSecure) have factories registered by the library's static initialization. CUDA-enabled builds register CudaHost and CudaDevice during CudaBootstrap::ensureRegistered. User-defined MemSpaces register their own factory after calling MemSpace::registerType / MemSpace::registerData.

Re-registering an ID replaces the previous factory. Thread-safe.
