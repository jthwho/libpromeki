Detailed Description

Rolling per-command telemetry collector for a single MediaIO.

Subscribes to a target MediaIO 's MediaIO::commandCompletedSignal and folds every numeric per-command stat (ExecuteDuration, QueueWaitDuration, BytesProcessed, plus any backend-specific keys) into a fixed-capacity WindowedStat ring keyed by the (MediaIOCommand::Kind, MediaIOStats::ID) pair the sample arrived under. Each window then exposes min / max / average / stddev / sample count via the WindowedStat accessors.

The collector is intentionally MediaIO-agnostic on the storage side — it does not pre-declare which keys to track. Any numeric Variant the framework or backend stamps onto a command's MediaIOCommand::stats becomes a window automatically the first time it is observed. Non-numeric entries (strings, enums, object types, invalid) are skipped silently — see WindowedStat::push(const Variant&) for the full promotion table.

Ownership
Inherits ObjectBase and uses ObjectBase parent-child ownership: pass the owning ObjectBase as parent to the constructor and the collector is destroyed alongside its parent. No shared ownership, no factory function, no std::shared_ptr.

The target is held via ObjectBasePtr so that the pointer auto-clears if the MediaIO is destroyed independently — the collector then becomes a no-op until setTarget rewires it.

Thread Safety
Inherits ObjectBase — thread-affine. Subscription to the MediaIO 's commandCompletedSignal uses the ObjectBase-owner Signal::connect overload, so per-command notifications are automatically marshalled onto the collector's EventLoop when the signal fires from a different thread (e.g. a strand worker). Every read and write to the window map therefore runs on the collector's own thread, so no internal mutex is required.

◆ setTarget()

void promeki::MediaIOStatsCollector::setTarget ( MediaIO *  io)

Switches to a new target (or detaches when io is nullptr).

Disconnects from the previous target's MediaIO::commandCompletedSignal (if any), clears the window map (windows from the old target are meaningless against the new one), and connects to the new target's signal. Pass nullptr to detach.

◆ setWindowSize()

void promeki::MediaIOStatsCollector::setWindowSize ( int  n)

Resizes every existing window in place.

When n is smaller than the current sample count each window drops its oldest samples first via WindowedStat::setCapacity. When n is zero every window is cleared and subsequent samples are dropped on the floor. Negative values clamp to zero.

Returns the live window map by const reference.

Walking the result is safe as long as no observed command lands on the collector concurrently — which is guaranteed when the caller runs on the collector's own EventLoop thread. Callers on other threads should marshal through the collector's loop.
