Detailed Description

One stamp request: write a 64-bit payload into a contiguous run of samples on a single channel.

firstSample and sampleCount are in payload-sample units (one unit per sample frame, the same units as AudioPayload::sampleCount). The codeword occupies the first BitsPerPacket * samplesPerBit() samples of the run; any additional samples up to sampleCount are zero-filled.
