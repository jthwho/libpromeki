Detailed Description

Heterogeneous string-keyed map of Variant values.

VariantMap is a Variant alternative — same shape as VariantList but keyed by String. Used for JSON-shaped objects, RPC argument bags, dynamic config payloads. As with VariantList, the underlying Map<String, Variant> lives behind a UniquePtr handle so the type is complete with known size even when Variant is incomplete.

Iteration
std::map storage is not contiguous, so map iteration is exposed via forEach and via keys rather than raw iterators. Per-key access is via value, find, and contains.
Thread Safety
Conditionally thread-safe — same contract as VariantList.
