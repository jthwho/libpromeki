Detailed Description

String-keyed map of Variant values.

Implemented as a thin subclass of Map<String, Variant> so that variant_fwd.h can declare it as an incomplete class — the same pattern used by VariantList. Used wherever a request needs to deliver a heterogeneous, named bag of values to a callee (RPC argument lists, dynamic config payloads, signal/slot cross-thread marshalling that already carries names alongside values).
