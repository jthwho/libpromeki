Detailed Description

template<typename T>
struct promeki::ToStringFormatter< T >

std::formatter helper that renders any type with a no-arg toString() returning a String.

Inherit a std::formatter specialization from this template to give a library type out-of-the-box support as a String::format argument. All standard string format specifiers (width, fill, alignment, precision) are inherited from std::formatter<std::string_view>, so users can write

String::format("[{:>16}]", uuid); // right-justified, width 16
String::format("ratio = {}", aspectRatio);
