Detailed Description

Describes a single color component for display and documentation.

Each color model has three color components (alpha is handled separately by Color). CompInfo provides the human-readable name and abbreviation, plus the native display range. Internally, components are stored normalized to 0.0-1.0, but the native range tells you what those values mean in conventional units. For example, HSV hue is stored as 0.0-1.0 but its native range is 0-360 degrees.
