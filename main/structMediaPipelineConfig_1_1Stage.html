Detailed Description

Declarative description of one MediaIO stage.

A stage is either a registered backend (identified by type — e.g. "TPG", "Converter") or an opaque filesystem path (identified by path with type empty) that is resolved at build time via MediaIO::createForFileRead / MediaIO::createForFileWrite. When both are set type wins and path is passed through to the backend as MediaConfig::Filename.
