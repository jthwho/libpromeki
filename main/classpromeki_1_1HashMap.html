Detailed Description

template<typename K, typename V>
class promeki::HashMap< K, V >

Unordered associative container wrapping std::unordered_map.

Provides a Qt-inspired API over std::unordered_map with consistent naming conventions matching the rest of libpromeki.

Thread Safety
Conditionally thread-safe. Distinct instances may be used concurrently; concurrent access to a single instance must be externally synchronized.
Template Parameters
KKey type (must be hashable).
VValue type.
Example
HashMap<String, int> cache;
cache.insert("width", 1920);
cache.insert("height", 1080);
int w = cache.value("width", 0); // 1920
