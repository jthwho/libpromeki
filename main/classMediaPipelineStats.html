Detailed Description

Aggregated statistics for a running MediaPipeline.

Holds a MediaIOStats snapshot for each stage, a rolled-up per-stage aggregate, and a PipelineStats bucket for counters that describe the drain layer itself (total fan-out, write retries, pipeline errors, back-pressure). The aggregate reduces the per-stage numbers as follows:

  • Counters (FramesDropped / FramesRepeated / FramesLate / QueueDepth / QueueCapacity / PendingOperations) are summed.
  • Throughput (BytesPerSecond / FramesPerSecond) is summed across stages — useful for total pipeline bandwidth.
  • Average latency / processing times are averaged over the set of stages that reported a non-zero value.
  • Peak latency / processing times use the maximum across stages.
  • LastErrorMessage takes the first non-empty message, prefixed with its stage name.

The snapshot round-trips through JSON (for logging / REST endpoints) and DataStream (for IPC).
