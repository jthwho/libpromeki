Detailed Description

template<typename T, typename Compare = std::less<T>>
class promeki::PriorityQueue< T, Compare >

Priority queue container wrapping std::priority_queue.

Provides a Qt-inspired API over std::priority_queue with consistent naming conventions matching the rest of libpromeki. Simple value type — no PROMEKI_SHARED_FINAL.

Thread Safety
Conditionally thread-safe. Distinct instances may be used concurrently; concurrent access to a single instance must be externally synchronized — typically by sharing the same Mutex with the producer-consumer surrounding it.
Template Parameters
TElement type.
CompareComparison function type (default: std::less<T>, which gives a max-heap where pop() returns the largest element).
