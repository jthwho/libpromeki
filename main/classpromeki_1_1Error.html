Detailed Description

Lightweight error code wrapper for the promeki library.

Encapsulates an error code from the Code enumeration and provides comparison operators, human-readable names, descriptions, and a mapping to the corresponding POSIX errno value when applicable.

Thread Safety
Distinct instances may be used concurrently. The class holds a single integer code; copying, assignment, and comparison are all trivially safe.
Example
Error err;
// Functions use Error* for fallible results:
auto val = someFunction(&err);
if(err.isError()) {
String msg = err.description();
}
// Direct comparison
if(err == Error::NotExist) { ... }

enum promeki::Error::Code

Error codes for the promeki library.

New codes should be added here whenever an existing code would be an imprecise fit. Prefer specific, descriptive codes over reusing generic ones. Codes that map to POSIX errno values are noted in error.cpp; domain-specific codes have no errno mapping.
