Detailed Description

Abstract base class for all I/O devices.

IODevice provides a uniform interface for reading and writing data to devices such as files, sockets, pipes, and in-memory buffers. All I/O device classes in promeki derive from this class.

Options
Device capabilities are expressed through options that subclasses register via registerOption(). Each option has a runtime-assigned type ID (allocated by registerOptionType()) and a Variant value. Client code queries and sets options by type ID:
if(dev.optionSupported(IODevice::DirectIO)) {
dev.setOption(IODevice::DirectIO, true);
}
auto [val, err] = dev.option(IODevice::DirectIO);
