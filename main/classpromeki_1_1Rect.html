Detailed Description

template<typename T>
class promeki::Rect< T >

2D rectangle defined by position and size.

A simple value type combining a Point<T,2> origin (top-left corner) with a Size2DTemplate<T> extent. Provides intersection, union, containment, and coordinate transformation operations.

Thread Safety
Distinct instances may be used concurrently. A single instance is conditionally thread-safe — const operations are safe, but concurrent mutation requires external synchronization.
Template Parameters
TThe component type (e.g. int, float, double).

bool hit = r.contains(15, 25); // true
