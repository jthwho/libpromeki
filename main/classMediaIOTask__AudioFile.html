Detailed Description

MediaIOTask backend for audio file formats.

Wraps the AudioFile subsystem (backed by libsndfile) to provide frame-based audio I/O through the MediaIO interface. Audio samples are chunked into frame-sized blocks according to the configured frame rate.

Supported formats: WAV, BWF, AIFF, OGG.

Config keys
Key Type Default Description
MediaConfig::Filename String — File path.
MediaConfig::FrameRate FrameRate 29.97 Frame rate for sample chunking.
MediaConfig::AudioRate float 48000 Audio sample rate Hz (required for writer).
MediaConfig::AudioChannels int 2 Audio channel count (required for writer).
