Detailed Description

Audio level in dBFS (decibels relative to full scale).

Simple value type that stores level as dBFS and converts to/from linear amplitude. 0 dBFS is full scale (linear 1.0), -6 dBFS is approximately half amplitude, and silence is negative infinity.

Example
AudioLevel level = AudioLevel::fromDbfs(-20.0);
double linear = level.toLinear(); // ~0.1
AudioLevel half = AudioLevel::fromLinear(0.5);
double db = half.dbfs(); // ~-6.02
