Detailed Description

Color science: color models, CIE colorimetry, and precise conversion.

Color and ColorModel classes for representing and converting colors across sRGB, Rec.709, Rec.601, Rec.2020, CIE XYZ, CIE L*a*b*, HSV, HSL, and YCbCr color models. Includes CIE chromaticity primitives (CIEPoint, XYZColor) and a standard observer wavelength table.

See Color Science for a conceptual introduction.
