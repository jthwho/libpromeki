Detailed Description

Content light level information (CTA-861.3 / ITU-T H.265 Annex D).

Describes the brightness characteristics of the actual content, complementing MasteringDisplay which describes the display used during grading. Together they give downstream tone-mappers the information needed to adapt HDR content to the viewer's display.

Field Unit Description
maxCLL cd/m² (nits) Maximum Content Light Level — brightest pixel in the entire stream.
maxFALL cd/m² (nits) Maximum Frame-Average Light Level — brightest per-frame average.

A default-constructed ContentLightLevel has both fields at zero, which means "unspecified" per CTA-861.3.

Example
ContentLightLevel cll(1000, 400);
assert(cll.maxCLL() == 1000);
assert(cll.maxFALL() == 400);
