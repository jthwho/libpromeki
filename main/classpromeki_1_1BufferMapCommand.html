Detailed Description

Command to map a Buffer for access from a specific MemDomain.

Built by Buffer::mapAcquire. The backend resolves the request by ensuring the buffer is accessible to target, populating any domain-specific output, and calling markCompleted. When target is MemDomain::Host, hostPtr is the pointer the caller can dereference until a matching BufferUnmapCommand runs.

Backends that expose richer per-domain outputs (an FPGA buffer index, an RDMA region key, a CUDA stream handle) subclass BufferMapCommand and add the field there; callers reach the typed field via BufferRequest::commandAs.
