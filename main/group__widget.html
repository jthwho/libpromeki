Detailed Description

Base widget classes, layouts, and events for the SDL GUI library.

Widget base class, Layout management (HBoxLayout, VBoxLayout, GridLayout, StackedLayout, FormLayout, FixedLayout), and widget events (PaintEvent, ResizeEvent, KeyEvent, MouseEvent).
