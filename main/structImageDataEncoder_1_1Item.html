Detailed Description

One scan-line band of an image to stamp with a single 64-bit payload.

firstLine and lineCount are in luma scan lines. For sub-sampled chroma planes the encoder writes any chroma row that overlaps the luma range — see the "Alignment" note in Image Data Encoder Wire Format.
