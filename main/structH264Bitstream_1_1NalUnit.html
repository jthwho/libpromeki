Detailed Description

Zero-copy view onto one NAL unit within a larger buffer.

view covers the NAL payload only — no start code and no length prefix. header0 is the first byte of that payload (convenient for H.264, where nal_unit_type = header0 & 0x1f). header1 is the second byte (meaningful for HEVC, whose NAL header is two bytes).
