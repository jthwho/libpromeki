Detailed Description

template<typename T>
class promeki::Promise< T >

Promise wrapping std::promise<T>.

Used to set a result that will be retrieved via the associated Future<T>. Move-only (non-copyable).

Thread Safety
Distinct instances may be used concurrently. A single Promise is intended to be set exactly once from a producer thread; the matching Future is observed from a consumer thread. The underlying std::promise/std::future synchronization makes the producer→consumer handoff safe.
