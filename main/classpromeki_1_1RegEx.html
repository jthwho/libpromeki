Detailed Description

Regular expression wrapper around std::regex.

Provides a simplified interface for pattern matching, searching, and extracting matches from strings using standard C++ regular expressions.

Thread Safety
A compiled RegEx is safe to share across threads for read-only operations (match, search, replace) — the underlying std::regex is internally const-safe. Recompiling (assigning a new pattern) requires external synchronization.
Example
RegEx re("(\\d+)x(\\d+)");
auto match = re.match("1920x1080");
if(match.hasMatch()) {
String w = match.captured(1); // "1920"
String h = match.captured(2); // "1080"
}
