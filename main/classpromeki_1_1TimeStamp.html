Detailed Description

A monotonic timestamp based on std::chrono::steady_clock.

Provides high-resolution time measurement, elapsed time queries, and thread sleep utilities. Uses steady_clock to guarantee monotonic progression regardless of system clock adjustments.

Thread Safety
Distinct instances may be used concurrently. A single instance is conditionally thread-safe — const operations are safe, but concurrent mutation requires external synchronization. Static helpers (now, sleep, etc.) are fully thread-safe.
Example
TimeStamp start = TimeStamp::now();
// ... do work ...
double elapsed = start.elapsedSeconds();
