Detailed Description

Output endpoint on a MediaNode that delivers Frame::Ptr to connected sinks.

Each MediaSource maintains a list of connected MediaSink::Ptr objects. When deliver() is called, the frame is pushed to every connected sink. sinksReadyForFrame() checks backpressure across all connected sinks.

MediaSource is managed via SharedPtr for lifetime management. It is an identity object (non-copyable) – the clone method should never be called.
