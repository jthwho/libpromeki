Registers a cirf-generated resource set at static-init time.

Expands to a file-scope object whose constructor calls Resource::registerRoot(&NAME##_root, PREFIX). Use it AFTER including the cirf-generated header so that <NAME>_root is in scope. The macro must appear at namespace scope.

Example
#include <promeki/resource.h>
#include "my_resources.h"
PROMEKI_REGISTER_RESOURCES(my_resources, "myapp")
// After this, ":/myapp/foo/bar.txt" resolves to my_resources_root's
// "foo/bar.txt".
