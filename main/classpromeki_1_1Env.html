Detailed Description

Provides access to process environment variables.

Static utility class wrapping standard environment variable operations. All methods return promeki::String for consistent integration with the rest of the library.

Thread Safety
The static read accessors (get, contains) are safe to call from any thread. set / unset modify the process environment via setenv / unsetenv, which on POSIX is not thread-safe with respect to other threads reading the environment — synchronize externally if mutating the environment after worker threads have started.
Example
String home = Env::get("HOME");
Env::set("MY_APP_DEBUG", "1");
bool exists = Env::contains("PATH");

static Map< String, String > promeki::Env::list ( const RegEx &  filter)
static

Returns environment variables whose names match a regex.

Iterates the entire process environment and returns only those variables whose name is matched by filter (using RegEx::search, so partial matches count).

Example
// All vars starting with PROMEKI_OPT_
auto opts = Env::list(RegEx("^PROMEKI_OPT_"));
