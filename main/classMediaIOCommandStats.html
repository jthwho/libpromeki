Detailed Description

Backend statistics query command.

Backends populate stats with whatever runtime metrics they track (frames dropped, queue depth, bytes/sec, latency, last error, etc.). See MediaIOStats for the standard key conventions. Backends are free to add backend-specific keys.

The default implementation returns Error::Ok with an empty stats.
