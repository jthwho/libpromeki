Returns the Duration between two TimeStamps.

Equivalent to a.value() - b.value() converted to the library's portable Duration type. Used wherever code wants "time since" or "time between" measurements without having to dip into raw std::chrono.

Parameters
aThe later TimeStamp.
bThe earlier TimeStamp.
Returns
a - b as a Duration. Can be negative if a precedes b.
