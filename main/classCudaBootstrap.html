Detailed Description

Process-wide CUDA bootstrap.

CudaBootstrap is the single entry point that wires CUDA-backed MemSpace::CudaDevice and MemSpace::CudaHost into the MemSpace registry. It is intentionally idempotent: call ensureRegistered from anywhere before you start allocating device or pinned-host memory and the first call does the real work; subsequent calls are no-ops.

The bootstrap does not select a device — callers pick one with CudaDevice::setCurrent if they care which GPU is used. When no device is currently selected, CUDA defaults to device 0.

Registers the CUDA-backed MemSpaces on first call.

Safe to call from any thread; registration runs under a static once-flag. Returns Error::NotImplemented when the library was built without CUDA support.
