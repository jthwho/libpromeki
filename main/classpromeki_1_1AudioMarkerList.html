Detailed Description

Ordered list of AudioMarker annotations on a payload.

AudioMarkerList is the value carried by Metadata::AudioMarkers — a producer-supplied annotation track that locates noteworthy regions within an audio payload (silence fills, concealed packet loss, glitches, discontinuities) and classifies each with an AudioMarkerType.

Markers are stored in the order the producer added them. The list does not enforce non-overlap or sortedness — overlapping regions are valid (e.g. a ConcealedLoss inside a larger SilenceFill) and the producer is free to append in arrival order. Consumers that need a sorted view should sort a copy.

String form

The list round-trips through a comma-separated list of AudioMarker::toString forms:

"SilenceFill@128+256, Discontinuity@384"

The empty list serializes to the empty string. Whitespace around the commas is permitted on parse.

Thread Safety

Plain value type with PROMEKI_SHARED_FINAL — copy / compare freely between threads, take a Ptr if shared ownership is desired. Concurrent mutation of one instance must be externally synchronized.

Sums the AudioMarker::length of every entry whose AudioMarker::type equals type. Overlapping regions are not deduplicated — a sample covered by two markers contributes twice.
