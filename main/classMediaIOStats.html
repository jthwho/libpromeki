Detailed Description

Statistics container for MediaIO backends.

A distinct VariantDatabase type for runtime stats reported by backends. Has its own StringRegistry so stats keys don't collide with config or param keys. Backends populate instances of this in executeCmd(MediaIOCommandStats&); users receive them from MediaIO::stats().

Standard key names are defined as static members; backends are free to add their own backend-specific keys for data not covered by the standard set.
