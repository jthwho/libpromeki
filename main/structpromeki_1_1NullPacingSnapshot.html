Detailed Description

Snapshot of NullPacingMediaIO internal counters.

Returned by NullPacingMediaIO::snapshot for tests and status displays. All counters are measured in frames; latency fields are in microseconds and reflect the time each frame spent inside the sink (arrival → discard). The snapshot is a value copy, taken under the sink's mutex, so it is safe to read from any thread regardless of strand activity.
