Detailed Description

JPEG VideoDecoder built on JpegImageCodec.

Symmetric counterpart to JpegVideoEncoder — incoming MediaPacket bytes are wrapped as a compressed Image and handed to JpegImageCodec::decode, with the resulting uncompressed Image queued for receiveFrame. The decoder's target uncompressed pixel description is set via MediaConfig::OutputPixelDesc; when unset, the codec's first declared decode target is used.

Registered against VideoCodec::JPEG.
