Detailed Description

JPEG VideoDecoder built on libjpeg-turbo.

Symmetric counterpart to JpegVideoEncoder — incoming MediaPacket bytes are handed to libjpeg-turbo for decode and the resulting uncompressed Image is queued for receiveFrame. The decoder's target uncompressed pixel description is set via MediaConfig::OutputPixelFormat; when unset, the first declared decode target for the input JPEG sub-format is used.

Registered against VideoCodec::JPEG.
