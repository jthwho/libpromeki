◆ mediaDescChanged

bool MediaIOCommandRead::mediaDescChanged = false

Set true when the backend wants to update the MediaDesc that MediaIO caches.

Used for variable-frame-rate or segmented streams whose format changes mid-playback. When true, MediaIO copies updatedMediaDesc into its cache, stamps Metadata::MediaDescChanged on the returned frame, and emits the descriptorChanged signal.
