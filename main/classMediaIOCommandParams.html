Detailed Description

Backend-specific parameterized command.

Carries an arbitrary operation name plus parameter and result containers, allowing backends to expose operations beyond the standard open/close/read/write/seek set. Examples: setting device gain, querying device temperature, triggering a one-shot capture, retrieving codec parameters.

Backends override executeCmd(MediaIOCommandParams &) and dispatch on name. Unrecognized names should return Error::NotSupported. Default implementation returns Error::NotSupported.
