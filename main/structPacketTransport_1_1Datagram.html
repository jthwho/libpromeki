Detailed Description

Describes one outbound datagram.

Identical in shape to UdpSocket::Datagram so that the UDP-backed transport can pass the list through unchanged. Other backends build their own mbuf / ring-buffer entries from these fields.

◆ txTimeNs

uint64_t PacketTransport::Datagram::txTimeNs = 0

Optional SCM_TXTIME nanoseconds-since-epoch deadline.

When zero the datagram is sent immediately. When non-zero and the transport supports it (see setTxTime()), the kernel / NIC holds the packet until the target time.
