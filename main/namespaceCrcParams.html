Detailed Description

Catalogue of standard CRC parameter sets.

Each entry is a constexpr CRC::Params instance — pass any one to a CRC constructor or to CRC::compute. Add new variants as needed; this is intentionally not exhaustive.

See also
CRC, crc8_smbus, crc8_autosar
