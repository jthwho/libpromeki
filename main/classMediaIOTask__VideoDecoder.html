Detailed Description

MediaIOTask backend that decodes compressed packets into uncompressed Frames.

MediaIOTask_VideoDecoder is the symmetric counterpart of MediaIOTask_VideoEncoder. It accepts a Frame whose compressed Images in Frame::imageList carry the encoded bitstream via Image::packet, hands each packet to a concrete VideoDecoder via submitPacket, and drains receiveFrame producing one output Frame per decoded image in Frame::imageList. Audio on the input Frame is forwarded unchanged so it stays synchronised on the same PTS.

The registered backend name is "VideoDecoder"; callers select a concrete codec in one of two ways:

  1. Explicit — set MediaConfig::VideoCodec in the config (e.g. "H264", "HEVC"). The decoder is created during open().
  2. Auto-detect — omit MediaConfig::VideoCodec. The task defers decoder creation until the first writeFrame() call, where it inspects the incoming MediaPacket::pixelDesc and resolves the codec via VideoCodec::fromPixelDesc.
Mode support

Only MediaIO::Transform is supported.

Config keys
Key Type Default Description
MediaConfig::VideoCodec VideoCodec (auto) Codec to use. When omitted the codec is detected from the first packet's MediaPacket::pixelDesc.
MediaConfig::OutputPixelDesc PixelDesc Invalid Desired uncompressed output format. Empty / Invalid means "use decoder's native".
MediaConfig::Capacity int 8 Output FIFO depth.
Example — explicit codec
MediaIO::Config cfg;
cfg.set(MediaConfig::Type, "VideoDecoder");
cfg.set(MediaConfig::VideoCodec, "H264");
cfg.set(MediaConfig::OutputPixelDesc, PixelDesc(PixelDesc::YUV8_420_SemiPlanar_Rec709));
MediaIO *dec = MediaIO::create(cfg);
dec->setExpectedDesc(compressedDesc);
dec->open(MediaIO::Transform);
dec->writeFrame(packetFrame);
Frame::Ptr decoded;
dec->readFrame(decoded);
dec->close();
