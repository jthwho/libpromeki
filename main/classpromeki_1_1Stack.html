Detailed Description

template<typename T>
class promeki::Stack< T >

LIFO stack container wrapping std::stack.

Provides a Qt-inspired API over std::stack with consistent naming conventions matching the rest of libpromeki. Simple value type — no PROMEKI_SHARED_FINAL (not typically shared or iterable).

Thread Safety
Conditionally thread-safe. Distinct instances may be used concurrently; concurrent access to a single instance must be externally synchronized. See Queue for a thread-safe FIFO alternative.
