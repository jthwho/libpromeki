inline

Parses the leading segment of a VariantLookup key.

Accepts names drawn from [A-Za-z0-9_@], an optional '[N]' suffix with a non-empty decimal index, and a dot-separated remainder. Returns false with Error::ParseFailed on empty input, empty names, empty indices, unterminated brackets, trailing dots, or any other structural anomaly. On success, err (when provided) is set to Error::Ok.

Parameters
keyThe full remaining key.
outPopulated with the parsed segment on success.
errOptional error output.
Returns
True on success, false on parse failure.
