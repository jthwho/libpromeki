Detailed Description

Well-known Enum type for chroma subsampling modes.

Mirrors JpegImageCodec::Subsampling in value and order. Used as the value type for MediaConfig::JpegSubsampling and anywhere else a simple 4:4:4 / 4:2:2 / 4:2:0 selection is needed.
