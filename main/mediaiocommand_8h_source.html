38// ============================================================================
39// Command hierarchy — all I/O operations are submitted as commands to the
40// MediaIO submit() virtual. Each command carries inputs (set by the public
41// API method that builds it) and outputs (set by the backend's
42// executeCmd()), so all data flow is lock-free.
