Detailed Description

PCM audio payload — linear PCM samples in the format described by the base AudioPayload's AudioDesc.

The plane list on the base carries one BufferView for interleaved PCM, or one per channel for planar PCM. Which layout applies is determined by the descriptor's AudioFormat (AudioFormat::isPlanar).

Each PCM audio payload represents a contiguous run of sampleCount samples per channel. Every PCM payload is trivially keyframe-able (decoding any sample does not require any earlier sample), so isKeyframe is overridden to always return true.

PcmAudioPayload is intentionally not final.

Example — allocating from a descriptor
AudioDesc desc(AudioFormat::PCMI_Float32LE, 48000, 2);
size_t samples = 1024;
auto buf = Buffer(desc.bufferSize(samples));
BufferView plane0(buf, 0, buf.size());
auto payload = PcmAudioPayload::Ptr::create(
desc, samples, plane0);
