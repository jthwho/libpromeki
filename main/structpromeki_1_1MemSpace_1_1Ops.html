Detailed Description

Function table for memory space operations.

The MemSpace wrapper handles trivial cases (zero-size allocations, null-pointer release/copy/fill) before dispatching, so backend lambdas may rely on:

  • alloc: a.size > 0.
  • release: a.ptr != nullptr.
  • copy: both src.ptr and dst.ptr non-null.
  • fill: ptr != nullptr.
