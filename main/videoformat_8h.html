X-macro defining the well-known video raster sizes.

Each entry has the form X(EnumID, CanonicalName, Width, Height). The macro is expanded in both VideoFormat::WellKnownRaster (enum generation) and in videoformat.cpp (registry data).

Only canonical names appear here. Additional parsing aliases ("NTSC", "PAL", "1080", "2160", "4KUHD", etc.) are listed in the source file.

◆ PROMEKI_WELL_KNOWN_VIDEO_FORMATS

#define PROMEKI_WELL_KNOWN_VIDEO_FORMATS

X-macro defining well-known video formats.

Each entry has the form X(EnumID, RasterID, FrameRateID, ScanMode, Flags).

  • EnumID — unqualified VideoFormat::WellKnownFormat identifier (e.g. Smpte1080p29_97).
  • RasterID — unqualified VideoFormat::WellKnownRaster id (e.g. Raster_HD).
  • FrameRateID — unqualified FrameRate::WellKnownRate id (e.g. FPS_29_97).
  • ScanMode — unqualified VideoScanMode static member (e.g. Progressive, Interlaced, PsF).
  • Flags — bitwise-OR of VideoFormat::WellKnownFormatFlag values describing the format family.

Expanded in both VideoFormat::WellKnownFormat (enum generation) and in videoformat.cpp (registry data).
