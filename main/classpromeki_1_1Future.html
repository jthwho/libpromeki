Detailed Description

template<typename T>
class promeki::Future< T >

Asynchronous result wrapping std::future<T>.

Returned by Promise<T>::future() and ThreadPool::submit(). Move-only (non-copyable).

Thread Safety
Distinct instances may be used concurrently. A single Future is intended to be observed from one consumer thread while its matching Promise is set from a producer thread; the underlying std::future synchronization makes this handoff safe. Calling get / wait / isReady on the same Future from multiple threads requires external synchronization.
Template Parameters
TThe result type.
Example
Promise<int> promise;
Future<int> future = promise.%future();
// In another thread:
promise.setValue(42);
// In the waiting thread:
int result = future.get(); // 42
